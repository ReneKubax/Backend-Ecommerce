# Backend-Ecommerce

**Node Js and Express**

Build locally:

**1. Clone Repo**

```bash
git clone https://github.com/ReneKubax/Backend-Ecommerce
```

**2. Install Dependencies**

```bash
npm i
```


**3. Run App**

```bash
npm run dev
```

**4. Use It**

You should be able to start working the backend.

Build locally Docker with Postgres:

**5. Run Docker Container**

```bash
docker-compose up
```
